Summary

Introduction to code in Python

I learnt the basics of Python. The first part was about the syntax of python, the data types, functions and even logical operators and operations. I learnt also about the different kinds of arrays. In the last part I learnt working with modules and a bit deeper about object-oriented programming.

How to create a simple webpage using HTML

I learnt that HTML is what defines the structure of a webpage. It contains different tags that are used for different displaying of specific content of a webpage. I also learnt how to create lists, both ordered and unordered, paragraphs and the different types of headings used in HTML.

Introduction to GO

I got to know that Go is used in a variety of applications today - server-side applications, DevOps, and command-line tools. I learnt of the different data types in Go which include: string, integer, boolean and float. And also learnt about Control Flow in Go.
